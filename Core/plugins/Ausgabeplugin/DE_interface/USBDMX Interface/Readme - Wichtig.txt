Übersicht:
----------
Beschreibung.txt - Allgemeine Beschreibung des Interfaces und der Softwareschnittstelle für PC-Programme
Aufbauanleitung.txt - Anleitung zum Aufbau der Interface Hardware
Bauteil-Liste.txt - Alle benötigten Bauteile
Firmware.txt - Informationen zum Generieren und Flashen der Firmware
usbdmx.txt - Informationen zur usbdmx.dll
Test.exe - Testsoftware für das fertig aufgebaute Interface
simple_usbdmx.txt - Stark vereinfachte DLL für den Zugriff auf ein Interface mit Code Beispielen in Visual Basic
usbdmx_example.cpp - Beispielcode für den Zugriff auf das Interface in C
Art-Net Show-Gate.exe - Dieses Tool ermöglicht aus einem PC mit Netzwerkanschluss und 1-4 Interfaces ein Art-Net Node zu machen
Art-Net Show-Gate AL.exe - "Art-Net Show-Gate" welches beim Start alle Einstellungen aus einer ini Datei lädt.
USBDMX_Debug.pdf - Ausführliche Debugging Informationen
USBDMX_TestAppVbNet.zip - Beispielprogramm in VBNet
