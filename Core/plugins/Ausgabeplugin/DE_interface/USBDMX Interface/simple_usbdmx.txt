usbdmxsi.dll - Schnittstellenbeschreibung
=========================================

-----------
Funktionen:
-----------

OpenInterface(DMXOutArray: PDMXArray; DMXInArray: PDMXArray; Mode: byte): DWORD;
CloseInterface: DWORD;

OpenInterface öffnet einen Link zum ersten angeschlossenen Interface im angegebenen Mode.
DMXOutArray und DMXInArray sind Buffer von 512 Byte welche Zugriff auf die Ausgabe- bzw. Eingabedaten ermöglichen.
CloseInterace schliesst den Link zum Interface.
